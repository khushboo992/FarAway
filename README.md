## Overview
**FarAway** is a responsive React-based travel packing list application built to help users efficiently organize and track items required for their trips. The application enables users to add items with quantities, manage packing status, and monitor overall progress in real time. This project demonstrates strong fundamentals in React, including state management, component-based architecture, and dynamic UI rendering.

## Features
- Add travel items with quantity
- Toggle packed and unpacked status
- Delete individual items from the list
- Clear the entire packing list with confirmation
- Sort items by input order, description, or packed status
- Real-time statistics showing packing progress
- Fully responsive design optimized for desktop, tablet, and mobile devices
- Clean and user-friendly interface
